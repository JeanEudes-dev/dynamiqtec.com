# Automatisez le MDX multilingue avec Gemma API et Next.js

Gérer plusieurs versions linguistiques peut être fastidieux. Automatisez les traductions MDX avec l'API Gemma.

## Configuration du CLI

```bash
npm install gray-matter node-fetch
```

## Script de traduction

- Lit les fichiers `.en.mdx`.
- Appelle le point de terminaison de traduction Gemma.
- Génère les fichiers `.fr.mdx` avec le frontmatter identique.

## Avantages

- Traductions cohérentes.
- Gain de temps.
- Intégration fluide dans les pipelines de build.
